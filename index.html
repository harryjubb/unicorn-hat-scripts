Unicorn HAT Scripts

Please find here some Open Source Python scripts for @pimoroni's Unicorn HAT. Included is unicorntools.py, a module with some extra helper functions for working with the HAT.

Documentation

Check out the scripts themselves to see what they do. Although they're in development, I endeavour to comment what's going on at each step.
